# App Store Connect screenshots — localized app surfaces

Generated by `automation/smoke/capture_app_store_surfaces_localized.py`.

The real in-app surfaces (not the onboarding tour), captured with realistic
seeded data on the **iPhone 17 Pro Max** simulator at **1320 × 2868 px** (the
iPhone **6.9" Display** size, which App Store Connect reuses for every other
iPhone size and localization).

6 surfaces per language, in filename order:

| File          | Surface                                            |
|---------------|----------------------------------------------------|
| `1-today`     | Today dashboard (check-in, Focus Three, Continue)  |
| `2-train`     | Train (planned session + readiness + history)      |
| `3-write`     | Write (capture inbox across stages)                |
| `4-career`    | Career (wins / impact / stories)                   |
| `5-home`      | Home (recurring tasks + household protocols)        |
| `6-digest`    | Today scrolled to Evening Reflection + Last Week    |

Chrome: light appearance, Apple-standard status bar (9:41, cellular bars,
**no Wi-Fi**, full battery).

## Fully localized content

Both the app **chrome** and the **seeded sample content** (focus items, notes,
training/career text, protocol steps, the digest insight) are localized in all
19 languages. The seed copy lives in a dedicated **`MarketingSeed.strings`**
table per locale — separate from the `Localizable.strings` UI catalog and its
native-review pipeline, so screenshot fixtures never disturb the shipped UI
strings. Source of truth: `Tools/marketing-seed-strings-generate.py`.

Seed translations are **LLM-drafted** (pending optional native review on the
marketing track); `OwloryUITestSupport.seedMarketing` reads them via
`NSLocalizedString(tableName: "MarketingSeed")` and falls back to English for
any blank entry.

## Folder → App Store Connect language

`en`→English (U.S.), `ar`→Arabic, `nl`→Dutch, `fr`→French, `de`→German,
`it`→Italian, `ja`→Japanese, `ko`→Korean, `nb`→Norwegian,
`pt`→Portuguese (Portugal), `pt-BR`→Portuguese (Brazil), `ru`→Russian,
`es`→Spanish (Spain), `sv`→Swedish, `zh-Hans`→Chinese (Simplified),
`zh-Hant`→Chinese (Traditional), `tr`→Turkish, `uk`→Ukrainian, `vi`→Vietnamese.

## Regenerating

```sh
python3 Tools/marketing-seed-strings-generate.py            # refresh seed-copy tables
python3 automation/smoke/capture_app_store_surfaces_localized.py             # all locales
python3 automation/smoke/capture_app_store_surfaces_localized.py --locales en,de,ar
```
