# App Store Connect screenshots — localized onboarding tour

Generated by `automation/smoke/capture_app_store_onboarding_localized.py`.

These are the 6 onboarding-tour pages (Welcome / Today / Train / Write / Career /
Home) captured on the **iPhone 17 Pro Max** simulator at **1320 × 2868 px**
(the iPhone **6.9" Display** size). Per App Store Connect, the 6.9" set is reused
for every other iPhone display size, so this is the only size you need to upload.

Chrome: light appearance, Apple-standard status bar (9:41, 4 cellular bars,
carrier, **no Wi-Fi**, full battery). The app UI is fully localized; sample
content inside the tour is the same translated copy Apple shows in the listing.

## How to upload

In App Store Connect → your app → the version → **App Previews and Screenshots**,
select the **6.9" Display** tab, pick a language from the localization dropdown,
and drag in the 6 PNGs from the matching folder below (in filename order
`1-…` → `6-…`).
